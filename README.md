# Development
```
$ npm install -g electron
$ electron -v
v1.7.9
$ electron .
```

# Create app file
```
$ npm install -g electron-packager
$ ./package.sh
```

# Create DMG
```
$ npm install -g electron-builder
$ electron-builder build
$ ll docs
```
